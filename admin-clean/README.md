# E-Commerce Admin Panel

Modern admin dashboard for managing e-commerce platform.

## Features

- Dashboard with analytics
- Product management (CRUD)
- Order management & tracking
- User management
- Category & subcategory management
- Coupon management
- Inventory tracking
- Returns & refunds
- Shipment tracking
- Reports & analytics
- Email templates
- Notifications
- Support tickets
- GDPR compliance
- Tax reports
- Audit logs

## Tech Stack

- React 18
- Vite
- React Router DOM
- Tailwind CSS
- Axios

## Installation

```bash
npm install
```

## Configuration

Create `.env` file:

```
VITE_API_URL=http://192.168.1.3:5000/api
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Default Admin Login

- Email: admin@example.com
- Password: admin123

## API Backend

This admin panel connects to the backend API. Make sure backend server is running on the configured URL.
